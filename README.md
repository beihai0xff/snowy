# snowy

Snowy 是一款面向高中生的 Web 端 AIGC 学习产品，首发聚焦：

- 高中知识检索
- 高中物理建模（推导 + 2D 图表 + 参数调节）

当前仓库已沉淀以下方案文档：

- [产品需求文档（PRD）](./docs/prd.md)
- [技术方案](./docs/tech-solution.md)

## 文档说明

### [docs/prd.md](./docs/prd.md)
- 偏研发落地的产品需求文档
- 覆盖产品目标、MVP 范围、核心功能、页面流程、接口边界、指标体系与里程碑

### [docs/tech-solution.md](./docs/tech-solution.md)
- 面向 Go 后端的技术方案
- 覆盖系统架构、统一索引、RAG 检索、多模型路由、物理建模与安全合规

## 当前约束
- 首发范围：知识检索 + 物理建模
- 数据源：课本与考纲、题库与讲义统一索引
- 模型策略：`gpt5` 主推理，`gemini3` 备选
- 物理可视化首发：2D 图表 + 参数调节，后续扩展动画仿真
